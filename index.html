Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • Add the source.
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Overkoalafied1", and install the OK1 repository.
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "OK1 Repository"
